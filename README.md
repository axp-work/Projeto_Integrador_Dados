PROJETO INTEGRADOR

1 - Tema: Análise de Vendas e Comportamento do Consumidor em E-commerce

2 - Integrante: Alex Sandro do Prado

3 - Objetivo da Análise:
Realizar uma análise de dados de e-commerce para identificar padrões de vendas, perfil dos clientes e principais indicadores de desempenho, apresentando os resultados em um dashboard no Power BI.

4 - Base de Dados:
Fonte: Olist Brazilian E-Commerce Dataset (Kaggle)
Descrição: Base de dados real com informações de aproximadamente 100 mil pedidos realizados em marketplace brasileiro entre 2016 e 2018. Contém dados de clientes, produtos, pagamentos, pedidos e avaliações.
Link: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

5 - Planejamento das Tarefas

* Configuração do repositório GitHub

* Download e importação da base de dados

* Limpeza e transformação dos dados

* Modelagem dos dados no Power BI

* Criação de métricas e indicadores

* Desenvolvimento do dashboard

* Documentação final

6 - Cronograma: 

* Criação do repositório

* Definição da base

* Limpeza e transformação

* Modelagem no Power BI

Transformações Previstas: 

* Tratamento de valores nulos

* Remoção de duplicatas

* Padronização de datas

Métricas e Visualizações no Dashboard:

- Métricas (KPIs):

* Faturamento total

* Quantidade de pedidos

* Ticket médio

* Avaliação média dos clientes

Visualizações:

* Evolução das vendas ao longo do tempo (gráfico de linhas)

* Vendas por categoria de produto (gráfico de barras)

* Formas de pagamento mais utilizadas (gráfico de pizza)

* Mapa com distribuição de vendas por estado

* Tabela com produtos mais vendidos

* Cards com os indicadores principais
